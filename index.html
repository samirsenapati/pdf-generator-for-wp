What's New in FinalPDF 2.0

üéØ Automatic Table of Contents

FinalPDF automatically generates clickable tables of contents for your PDFs. Perfect for knowledge bases, documentation sites, and long-form content. Extracts H1-H6 headings and creates beautiful, navigable PDFs with page references.

Key Features

Table of Contents

Automatically extracts headings and generates clickable TOC with page numbers

Clickable Navigation

Each TOC entry links directly to the corresponding section in the PDF

Custom Templates

Multiple PDF layouts with customizable headers, footers, and styling

Bulk Export

Generate PDFs for multiple posts, pages, or products at once

WooCommerce Ready

Create product PDFs, invoices, and receipts with full HPOS compatibility

Access Control

Role-based PDF visibility and download permissions

Installation Instructions

  1. Download the plugin using the button above (finalpdf-2.0.6.zip)
  2. Go to your WordPress Admin ‚Üí Plugins ‚Üí Add New ‚Üí Upload Plugin
  3. Choose the file you just downloaded (finalpdf-2.0.6.zip)
  4. Click "Install Now" and wait for installation to complete
  5. Activate the plugin from the confirmation screen
  6. Configure settings under FinalPDF menu in WordPress admin
  7. Enable TOC and customize depth, title, and placement
  8. Test it out by generating a PDF from any post or page

Requirements

  • WordPress 5.5.0 or higher
  • PHP 7.4 or higher
  • 256M memory recommended
  • WooCommerce 5.2.0+ (optional, for eCommerce features)

Table of Contents Settings

After installation, configure your TOC preferences:

  • Enable/disable automatic TOC generation
  • Set heading depth (H1-H2 only or H1-H6 for full hierarchy)
  • Customize TOC title and styling
  • Choose TOC placement (after cover page, before content)
  • Configure page number display format
