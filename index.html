About This Plugin

PDF Generator for WordPress allows you to convert posts, pages, and products into PDF files for easy sharing and distribution. The plugin integrates seamlessly with WordPress and WooCommerce.

Installation Instructions

To use this plugin in a production WordPress environment:

  1. Download the plugin files from the wordpress-site/wp-content/plugins/pdf-generator-for-wp directory
  2. Upload to your WordPress installation's wp-content/plugins/ directory
  3. Activate the plugin from the WordPress admin panel
  4. Configure settings under PDF Generator in the WordPress admin menu

Technical Details

  • PHP Version Required: 7.4 or higher
  • WordPress Version: 5.5.0 or higher
  • WooCommerce: 5.2.0 or higher (optional)
  • PDF Library: DomPDF (included in package/lib/dompdf/)

Plugin Structure

  • admin/ - Admin interface and backend functionality
  • public/ - Frontend display and user-facing features
  • includes/ - Core plugin classes and functions
  • package/lib/dompdf/ - PDF generation library
  • languages/ - Translation files
